Blackbox:

BlackBox é uma IA que te ajuda a criar códigos com as linguagens de programação que você mandar o APP fazer, por exemplo: Caso você mande o APP criar um app com HTML, JavaScript e CSS o app vai fazer o código de acordo oque você mandou. Resumidamente, o app consiste em criar códigos com a linguagem de programação que você mandar a IA fazer, Pode usar Java, JavaScript, Python, HTML ou a linguagem que você mandar o APP usar.
